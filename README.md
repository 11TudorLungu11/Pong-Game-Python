# Pong Game - Python

## Overview

This project is a classic Pong game developed in Python using the Pygame library.

The game features both Singleplayer and Multiplayer modes, multiple match lengths and a simple menu system for game navigation. It was created to practice object-oriented programming, game logic and event handling.

---

## Features

- Main menu
- Singleplayer mode
- Multiplayer mode
- Three match lengths:
  - Short (2 points)
  - Medium (5 points)
  - Long (10 points)
- AI opponent in Singleplayer
- Two-player local mode
- Random ball color after paddle collisions
- Victory screen displaying the winner
- Keyboard controls
- Score tracking

---

## Controls

### Main Menu

- **1** → Singleplayer
- **2** → Multiplayer
- **ESC** → Quit

### Singleplayer

Player controls:

- **W** → Move Up
- **S** → Move Down

Computer controls the second paddle automatically.

### Multiplayer

Player 1:

- **W** → Up
- **S** → Down

Player 2:

- **↑** → Up
- **↓** → Down
